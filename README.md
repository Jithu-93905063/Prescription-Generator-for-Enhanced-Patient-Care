# 🏥 Prescription Generator for Enhanced Patient Care

## 📌 Overview
This is a full-stack web application that allows users to input symptoms and receive relevant health suggestions. The system also enables users to connect with registered doctors by sharing their symptoms and receiving prescriptions through the platform.

---

## ⚙️ Features
- User registration and login system  
- Symptom input and processing  
- Health suggestions based on user input  
- Option to send symptoms to a registered doctor and notify through via email  
- Doctor can review and respond with prescriptions through the application  
- Secure data handling and storage  

---

## 🏗️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP 
- **Database:** MySQL   

---

## 💡 How It Works
1. User logs into the system  
2. Enters symptoms through the interface  
3. Backend processes input and generates relevant suggestions  
4. User can choose to send details to a registered doctor  
5. Doctor reviews and responds with a prescription via the platform
